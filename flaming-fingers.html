Flamin' Fingerz

A two-player typing speed game built in Java Swing. Pick a difficulty level, enter player names, hit Start, and type as fast as you can. Tracks WPM, accuracy, and time — and tells you who won.

Built by Atharv Gupta, Paarth Jain & Joshua Sahaya Arul  ·  Summer CS, 2016

Play

Loading Java runtime…

Runs via CheerpJ (Java → WebAssembly). First load takes ~10 seconds.

How to play

1 Pick a level — Level 1 is easy, Level 3 throws Latin and high-level literature at you. Test mode is short, good for a quick go.
2 Choose 1 or 2 players and optionally enter names. In 2-player mode, Player 2 gets the same paragraph after Player 1 finishes.
3 Click Begin, then click Start on the game board. A READY / SET / GO countdown kicks off — start typing immediately.
4 Type each word and hit space to advance. A red flash means a wrong word — fix it. When the paragraph is done, hit Scores to see results.

Built in jGRASP for a high school computer science final project. Java 8, Swing, javax.sound. No libraries beyond the JDK.
